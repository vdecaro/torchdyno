============
**Overview**
============

TorchDyno is a PyTorch library for non-conventional recurrent models for time
series — Reservoir Computing and other dynamical-systems models.

Installation
============

Install the ``torch`` build for your platform first (see the
`PyTorch install guide <https://pytorch.org/get-started/locally/>`_), then:

.. code-block:: bash

    pip install torchdyno

Optional example datasets that require extra dependencies are available via the
``datasets`` extra:

.. code-block:: bash

    pip install "torchdyno[datasets]"
